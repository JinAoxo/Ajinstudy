  • 인증서를 최초 1회 등록하시면 홈페이지 이용 시 편리하게 로그인할 수 있습니다.
  • 홈페이지 회원이 아닌 경우 먼저 회원 가입을 하셔야 합니다. 회원가입

인증서 등록 절차

  • 현대카드 홈페이지에서는 인증서 직접 발급 및 재발급이 불가하므로 해당 업무는 각 은행 및 증권사를 이용하시기 바랍니다.
  • 2014년 8월 7일 시행 개인정보보호법 개정으로 주민등록번호 수집·이용이 금지됨에 따라 주민등록번호 입력을 필요로 했던 인증서 로그인 절차를 새롭게 수정하였습니다.
  • 인증서 등록 후 추가적인 주민등록번호 인증 절차는 생략함으로써 회원님의 주민등록번호 노출을 최소화하고자 합니다.
  • STEP1 개인정보 입력 후 등록 버튼 클릭
  • STEP2 등록할 인증서 선택 후 암호 입력
  • STEP3 인증서 등록 완료 후 확인 버튼 클릭
  • STEP4 로그인 화면에서 주민등록번호 입력 없이 인증서 로그인 가능

인증서 삭제 절차

  • STEP1 개인정보 입력 후 삭제 버튼 클릭
  • STEP2 삭제할 인증서 선택 후 암호 입력
  • STEP3 인증서 삭제 완료
  • STEP4 인증서 삭제 시 아이디 로그인으로 홈페이지 이용 가능

인증서 소개 및 발급 안내

인증서란?

인터넷 상에서 발생하는 모든 전자거래를 안심하고 사용할 수 있도록 해주는 사이버 인감증명서입니다.

인증서 사용 안내

온라인쇼핑몰에서 신용카드로 30만원 이상 결제 시 인증서를 꼭 이용하세요.
회원님의 안전한 온라인쇼핑을 위해 2005년 11월 1일부터 본인 확인을 위한 절차로 인증서 이용 의무화가 시행되고 있습니다.

  • 1.인증서 신규 사용자 은행/보험용
    • 은행 방문 후 인터넷 뱅킹 신청
    • 해당 은행 웹 접속하여 인증서 등록 절차 거쳐 인증서 다운
  • 2.인증서 기존 사용 은행/보험용
    • 2006년 8월 1일 이전 발급 : 해당 은행 재발급 후 사용 가능
    • 2006년 8월 1일 이후 발급 : 바로 사용 가능
  • 인증서가 없으신 분은 가까운 거래은행에서 발급받으실 수 있습니다.
  • 공인인증기관으로는 금융결제원, 코스콤, 한국무역정보통신, 한국전산원, 한국전자인증, 한국정보인증 등이 공인인증기관으로 지정되어 있습니다.

인증서 FAQ

  • 최근 정부의 주민등록번호 수집 이용 최소화 정책에 따라, 주민등록번호를 직접 입력하는 부분을 최소화 하기 위하여 해당 프로세스를 적용하였습니다.
    개인정보보호법 개정(2014. 8. 7 시행)으로 법령에서 허용된 경우 외에는 모든 공공·민간 기업의 주민등록번호 수집·이용이 금지되었습니다.

  • 인증서를 재발급 받으셨거나, 인증서 유효기간이 만료된 경우 인증서 등록/삭제 페이지에서 인증서를 재등록하셔야만 합니다.
    현대카드는 인증서 발급기관이 아니므로, 인증서를 발급하거나 재발급해 드릴 수 없습니다.

  • 현대카드 홈페이지에서 인증서 최초 1회 등록을 완료하셨다면, 모바일 웹/앱에서는 별도 등록이 필요하지 않습니다.
    모바일에서 인증서 로그인을 사용하시려면 홈페이지에서 반드시 등록 절차를 완료해 주셔야 합니다.
